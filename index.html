Como criar um token Fine-grained:
  1. Acesse github.com → Settings → Fine-grained tokens
  2. Em Resource owner, selecione o owner do repositório
  3. Em Repository access, escolha Only selected → ishs_programme
  4. Em Permissions → Contents, selecione Read and write
  5. Gere e cole o token aqui
